# 诗经数据集标注工作流

本文档描述从原始文献到结构化 JSON 的完整流程，供协作者参考。

---

## 概览

```
ctext.org 原文
     ↓
手工整理为 txt（输入文件）
     ↓
AI 提取结构化注释
     ↓
人工校对
     ↓
写入 JSON 数据集
     ↓
render.py 渲染为 Markdown（可读性审阅）
```

---

## 第一步：获取原文

来源：[毛诗正义（武英殿十三经注疏本）](https://ctext.org/wiki.pl?if=gb&res=871901)

逐篇复制原文，整理为如下 txt 格式。每句以 `---` 分隔：

```
句子id：shijing_guofeng_zhounan_getan_s001
校勘文：
葛之覃兮，施于中谷，维叶萋萋
注释：
传：兴也。葛所以为絺綌，女功之事烦辱者。
覃，延也。施，移也。中谷，谷中也。萋萋，茂盛貌。
笺云：葛者，妇人之所有事也……
○施，毛以豉反，郑如字，下同。
萋切兮反。
---
```

**id 命名规则：**

```
shijing_{卷}_{子卷}_{篇}_s{NNN}
```

| 卷 | 拼音 |
|---|---|
| 国风 | guofeng |
| 小雅 | xiaoya |
| 大雅 | daya |
| 颂 | song |

子卷示例：周南 → zhounan，召南 → zhaonan

句序 `s001` 从 001 起，每篇独立编号。

---

## 第二步：AI 提取注释

将 txt 输入 DeepSeek R1（专家模式），使用以下提示词：

````
你是一个专业的先秦文献注释助手，熟悉《诗经》及其传统注疏。

## 任务
给定一句《诗经》校勘文及其注疏原文，提取所有被注释的字、词、句，
连同注释内容一并输出为 JSON 数组。

## 输入格式
依次提供若干条数据，每条包含：
- 句子id
- 校勘文（该句原文）
- 注释（包含毛传、郑笺、音读、校勘等注疏原文）
- 今译（现代汉语翻译，供填写白话字段参考）
- 注音（汉语拼音，供对应 anchor 参考）

## 规则

1. **anchor** 只能是校勘文的连续子串，不得使用注疏原文中的文字。
   - 校勘文末尾标点不计入 anchor，只取文字部分。
   - 若注释无法对应具体字词，粒度填"句"，anchor 填完整校勘文（不含标点）。

2. **内容** 必须是注疏原文的原文摘录，不得意译或改写。

3. **毛传和郑笺分开列**，每条注释对应一个 anchor。

4. **注释类型判断：**
   - 以"传："或无标识的字词训释 → `训诂`（毛传）
   - 以"笺云："引导 → `笺注`（郑玄）
   - 内容为制度性铺陈（礼制、官职、服饰等级等），非单字训释 → `句义`
   - 以"○"引导的反切、直音 → `音韵`
   - 版本异文、案语 → `校勘`

5. **id** 每句独立编号，从 `n001` 开始递增。

6. **span**：若同一 anchor 在校勘文中出现多次，填写 `[start, end)` 字符偏移，
   标点计入偏移。只出现一次则不填。

7. **文献id** 固定填 `src001`。

8. **同一 anchor 多条注释** 允许，注释类型不同即可分开列。

9. **输出**：只输出 JSON 数组，用 markdown 代码块包裹。

## 输出 Schema

{
  "id": "n001",
  "anchor": "覃",
  "粒度": "字",          // "字" | "词" | "句"
  "文献id": "src001",
  "注释类型": "训诂",    // "训诂"|"笺注"|"句义"|"考证"|"音韵"|"校勘"
  "内容": "覃，延也。"
}
````

---

## 人工校对（必须）

AI 输出后，逐条检查以下几点：

| 检查项 | 常见错误 |
|---|---|
| anchor 是否为校勘文子串 | 混入注疏文字；标点误入 |
| 内容是否为原文摘录 | 意译、合并、截断 |
| 毛郑是否分层正确 | 无标识条目误归笺注 |
| 句义判断 | 制度性铺陈误标为训诂 |
| 同字多注 | 音韵条目漏标或与训诂合并 |
| span 填写 | 重复 anchor 未填偏移 |

校对完成后，将注释数组填入对应句子的 JSON 文件。

---

## 填写今译与注音（可选）

在句子对象的顶层填写：

```json
"今译": "葛藤延伸啊，蔓延到谷中，叶子长得茂盛。",
"注音": "gé zhī tán xī shī yú zhōng gǔ wéi yè qī qī"
```

- 今译以程俊英《诗经译注》为准
- 注音以程俊英注音为准；破读或两读处在对应注释条目加 `"注音_存疑": true`

---

## 渲染审阅（可选）

```bash
python tools/render.py data/feng/zhounan/getan.json getan_rendered.md
```

生成 Markdown 供非技术协作者审阅，无需接触 JSON。

## 快速参考

| 字段 | 必填 | 说明 |
|---|---|---|
| id | ✅ | 句内 n001 起，每句重置 |
| anchor | ✅ | 校勘文子串，不含标点 |
| span | ❌ | anchor 重复出现时填 |
| 粒度 | ✅ | 字 / 词 / 句 |
| 文献id | ✅ | 固定 src001 |
| 注释类型 | ✅ | 见上表 |
| 内容 | ✅ | 原文摘录，不改写 |
| 注音_存疑 | ❌ | 破读或争议读音时标 true |
